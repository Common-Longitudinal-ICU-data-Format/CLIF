# mCIDE: minimum Common ICU Data Elements for CLIF

_To learn more about the CLIF mCIDE, explore the [About mCIDE](https://clif-icu.com/mCIDE) page on the [CLIF website](https://clif-icu.com/)._

Each subfolder contains mCIDE category files corresponding to the category variables in the respective v2.1 CLIF Beta tables:

- **`adt/`** 
    - hospital_type
    - location_category
    - location_type
- **`code_status/`**  
    - code_status_category
- **`crrt_therapy/`** 
    - crrt_mode_category
- **`hospitalization/`** 
    - admission_type_category
    - discharge_category
- **`hospital_diagnosis/`** 
    NA 
- **`labs/`** 
    - lab_category
    - lab_order_category
- **`medication_admin_continuous/`** 
    - med_category
    - med_route_category 
    - mar_action_category and mar_action_group
- **`medication_admin_intermittent/`** 
    - med_category
    - med_route_category 
    - mar_action_category and mar_action_group
- **`microbiology_culture/`** 
    - fluid_category
    - method_category
    - organism_category
    - organism_group
- **`microbiology_susceptibility/`** 
    - antimicrobial_category
    - susceptibility_category
- **`patient/`**
    - race_category
    - ethnicity_category
    - sex_category
    - language_category
- **`patient_assessments/`** 
    - assessment_category
- **`patient_procedures/`** 
    - procedure_code_format, procedure_code, and proc_name
- **`postion/`** 
    - position_category
- **`respiratory_support/`** 
    - device_category
    - mode_category
- **`vitals/`** 
    - vital_category
